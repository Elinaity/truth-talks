# TabBar Icons

需要添加以下图标文件（建议尺寸：81×81px）：

- home.png (首页未选中)
- home-active.png (首页选中)
- post.png (发布未选中)
- post-active.png (发布选中)
- mine.png (我的未选中)
- mine-active.png (我的选中)

可从以下资源获取免费图标：
- https://www.iconfont.cn/
- https://www.flaticon.com/
- https://iconscout.com/

建议风格：简约线性图标，颜色 #888888（未选中）和 #6B5B95（选中）
